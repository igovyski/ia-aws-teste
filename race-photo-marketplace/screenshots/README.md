# Screenshots

Este diretório contém as capturas de tela da aplicação em funcionamento.

Para adicionar screenshots:

1. Execute a aplicação usando `start.bat` ou `python src/app.py`
2. Acesse http://localhost:5000
3. Tire screenshots das principais telas:
   - home.png - Página inicial
   - search.png - Busca de fotos
   - cart.png - Carrinho de compras
   - admin.png - Painel administrativo

## Exemplo de uso:

1. **Página Inicial**: Mostra eventos disponíveis e campo de busca
2. **Busca**: Digite um número do peito (ex: 123) para encontrar fotos
3. **Login**: Use admin@example.com / admin123 ou cliente@example.com / cliente123
4. **Carrinho**: Adicione fotos e simule uma compra
5. **Admin**: Crie novos eventos (apenas para usuários admin)
